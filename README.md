# Sleep Health & Lifestyle Analysis

## Overview
A data analysis project exploring relationships between sleep health, stress levels, physical activity, and lifestyle factors. Uses machine learning to predict sleep duration based on health metrics.

## Key Results
- Built Linear Regression model to predict sleep duration from health metrics
- Analyzed correlations between stress, physical activity, heart rate and sleep quality
- Identified key lifestyle factors that significantly impact sleep duration
- Created comprehensive visualizations of sleep health patterns

## Tools & Technologies
- Python — Pandas, NumPy, Scikit-learn
- Visualization — Matplotlib, Seaborn
- Machine Learning — Linear Regression
- Analysis — Correlation analysis, Statistical distributions

## Dataset
- Features: Sleep Duration, Stress Level, Physical Activity Level, Heart Rate, Daily Steps
- Target: Sleep Duration prediction

## How to Run
pip install pandas numpy scikit-learn matplotlib seaborn
jupyter notebook Sleep_Health_Analysis.ipynb

## Key Findings
- Stress level has strong negative correlation with sleep duration
- Physical activity positively impacts sleep quality
- Heart rate and daily steps are significant predictors of sleep duration
